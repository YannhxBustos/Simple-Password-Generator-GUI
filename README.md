# Simple Password Generator

A simple yet powerful password generator written in Java that allows you to create secure passwords with customizable options. This program requires JDK 25 to run.

## Features

- **Customizable password length**: Set the exact number of characters for your password
- **Flexible character options**:
  - Uppercase letters (A-Z)
  - Lowercase letters (a-z)
  - Numbers (0-9)
  - Special characters (!, @, #, $, %, ^, &, *, etc.)
- **Command-line interface**: Easy to use with simple parameters
- **Cross-platform**: Works on any system that supports JDK 25

## Requirements

⚠️ **This program requires JDK 25 to run properly.** ⚠️

- Java Development Kit (JDK) 25 or higher
- Java Runtime Environment (JRE) 25 (if only running the compiled JAR)

**Important**: The program will not work with older JDK/JRE versions due to features and APIs specific to JDK 25.

## Getting Started

### Option 1: Using the Pre-built JAR File

1. Download the latest `PasswordGenerator.jar` file from the [Releases](https://github.com/YannhxBustos/Simple-Password-Generator-GUI/releases) section
2. Ensure you have JDK 25 installed on your system
3. Run the program using the following command:

1 - Use the terminal.
2 - Make sure you have JDK 25 installed, and double-click the .jar file.
